## What's included

OCA Web Addons covers six main areas of the Odoo web interface:

<Steps>
  <Step title="Choose your addons">
    Browse the addon groups above to find the features you need. Each addon is independent and installable on its own.
  </Step>
  <Step title="Install via pip or source">
    Install addons from PyPI using `pip install odoo-addon-<name>` or clone the repository and add it to your Odoo addons path.
  </Step>
  <Step title="Activate in Odoo">
    Go to **Settings → Apps**, search for the addon by name, and click **Install**. Most addons activate immediately with no additional configuration.
  </Step>
  <Step title="Configure as needed">
    Some addons — like `web_environment_ribbon`, `web_favicon`, and `web_session_auto_close` — have optional system parameter configuration documented in their respective pages.
  </Step>
</Steps>

## Compatibility

<Note>
  All addons in this documentation target **Odoo 18.0**. Earlier versions (16.0, 17.0) are maintained on separate branches in the [GitHub repository](https://github.com/OCA/web).
