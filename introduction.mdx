## What is OCA?

The **Odoo Community Association** (OCA) is a nonprofit organization whose mission is to support the collaborative development of Odoo features and promote its widespread use. OCA coordinates open-source development of Odoo modules, maintains quality standards across hundreds of repositories, and provides a neutral home for community contributions.

All OCA modules are developed transparently on GitHub, reviewed by peers, and released under OSI-approved licenses (AGPL-3.0 or LGPL-3.0).

## What is this repository?

The [OCA/web](https://github.com/OCA/web) repository is the community's primary collection of **web client UI addons** for Odoo 18.0. It contains 60+ modules that extend, customize, or fix Odoo's browser-based frontend without requiring Odoo Enterprise.

Key characteristics:

- Every addon is **independently installable** — install only what you need.
- All addons target **Odoo 18.0** (branch `18.0`).
- Continuous integration runs pre-commit checks, unit tests, and coverage reporting on every pull request.
- Translations are managed through [Weblate](https://translation.odoo-community.org/engage/web-18-0/).

<Note>
  These addons work with the Odoo Community Edition. Some addons explicitly exclude `web_enterprise` to avoid conflicts with the Enterprise client.
</Note>
